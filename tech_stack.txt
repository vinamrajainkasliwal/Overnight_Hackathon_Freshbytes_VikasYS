# Tech Stack – E-Farmer Subsidy Leakage Detection System

----------------------------------------------------
FRONTEND
----------------------------------------------------
• HTML  
• CSS  
• JavaScript  

Purpose:
• User interface for farmer registration, dealer claims, and dashboard
• Display E-Farmer ID card and fraud alerts

----------------------------------------------------
BACKEND
----------------------------------------------------
• Python (Flask Framework)

Purpose:
• Handle data storage, entitlement calculations, and fraud detection logic
• Provide APIs for frontend communication

----------------------------------------------------
DATABASE / STORAGE
----------------------------------------------------
• JSON & CSV files

Used For:
• Farmer registration data (farmers.json)
• Dealer transaction logs (transactions.json)
• Entitlement rule mapping (entitlement_rules.json)
• Flagged suspicious cases (flagged_cases.json)

----------------------------------------------------
AI INTEGRATION (Prototype)
----------------------------------------------------
• Land photo upload with GPS metadata (mock verification)
• Neighbor cross-verification concept
• ML-based land & crop validation planned in future scope

----------------------------------------------------
DEV TOOLS
----------------------------------------------------
• GitHub – Version control, commits for each checkpoint, submission repository
• Browser console – Quick testing of logged data

----------------------------------------------------
REASON FOR THIS STACK
----------------------------------------------------
• Simple and fast to build overnight
• Easily demonstrates identity, entitlement & fraud detection flow
• Allows us to simulate Aadhaar/GPS/ML integrations for hackathon
